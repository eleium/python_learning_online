操作目的	   命令	                示例 &                   说明
创建文件夹	mkdir 文件夹名	    mkdir python_projects →  创建名为 python_projects 的文件夹
	        mkdir -p 多级文件夹	mkdir -p a/b/c           →一次性创建 a/b/c 三级文件夹（-p 自动创建父目录）
创建空文件	touch 文件名	        touch test.py           → 创建空的 test.py 文件touch a.txt b.csv → 同时创建多个文件
查看当前路径	pwd	                输 pwd                   → 输出 /home/elei（告诉你现在在哪）
查看文件    / 文件夹	ls	输 ls → 列出当前目录下的所有文件 / 文件夹ls -l → 显示详细信息（大小、权限）ls -a → 显示隐藏文件（以。开头的）
进入文件夹	cd 文件夹路径	   cd python_projects       → 进入当前目录下的 python_projects 
            cd ..            → 回到上一级目录（.. 代表父目录）cd /mnt/d → 直接跳转到 D 盘
回到家目录	cd 或 cd ~	无论在哪，输 cd 立刻回到 elei@T440:~$ 位置


创建：mkdir（文件夹）、touch（空文件）；
查看 / 跳转：ls（看内容）、pwd（看位置）、cd（跳路径）；
编辑 / 保存：nano 文件名 → Ctrl+O 保存 → Ctrl+X 退出；
删除：rm（文件）、rm -r（文件夹）（谨慎用！）；
复制 / 移动：cp（复制）、mv（移动 / 重命名）。
