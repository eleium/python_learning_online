语法结构		缩进要求
if-else		if、elif、else 必须同级对齐，代码块缩进一级

while-else		while和else必须同级对齐，循环体缩进一级

elif		与if同级对齐（不是if的子句，而是平行条件）


age = 18
if age < 18 :
    print('未成年')
elif age == 18:
    print('刚成年')
else:
    print('成年人')


count = 0
while count < 3:
    print(count)
    count+=1
else:
    print('循环结束')
